XliffTemp

XliffTemp.xsl — Unescapes escaped markup in an XLIFF file.

Parameters

None.

Input

An XLIFF document containing escaped inline markup.

Output

A temporary, invalid XLIFF document containing unescaped inline markup.

Description

The stylesheet copies the structure and content of the input document to the output document. Any escaped inline markup within the input document is unescaped.

The output of this stylesheet is intended to be used as input for XliffTag.xsl, which turns the inline markup into valid XLIFF g, ph, x and mrk elements.
